# 🔐 KeyKeeper

A full-stack password management web application built using **React (Vite)** for the frontend and **Node.js + Express + MongoDB** for the backend.

---

## 🌐 Live Demo

- 🔗 Frontend: https://key-keeper-xi.vercel.app  
- 🔗 Backend: https://keykeeper-sxda.onrender.com  

---

## 📌 Features

- 🔑 Secure password storage
- 👤 User authentication (JWT-based)
- 🔒 Encrypted password handling
- 📋 Copy to clipboard functionality
- 🌙 Clean and responsive UI
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT (Authentication)
- bcrypt.js (Password hashing)
